Display: Inline-Block

Compreenda a propriedade CSS que combina características de elementos inline e block

O que é display: inline-block?

O display: inline-block; é um valor da propriedade CSS display que combina características dos elementos inline e block.

.elemento {
  display: inline-block;
}

Como funciona?

Elementos com display: inline-block; possuem as seguintes características:

Elementos Inline

  • Ficam na mesma linha
  • Não aceitam width e height
  • Aceitam apenas padding e margin horizontais

Quando usar inline-block?

Use display: inline-block; quando precisar de elementos que permaneçam na mesma linha, mas que também precisam ter dimensões definidas (largura e altura).

Listas horizontais

Para criar menus de navegação horizontal onde cada item precisa ter tamanho específico.

Grades de produtos

Para exibir produtos ou elementos em grade que precisam ficar na mesma linha.

Elementos lado a lado

Para colocar elementos lado a lado com controle sobre suas dimensões.
