O que é Display Flex?

O display: flex é um método de layout em CSS que facilita o design de estruturas responsivas e a organização de elementos dentro de um container.

Quando aplicamos display: flex a um elemento, ele se torna um flex container, e seus filhos diretos se tornam flex items.

.container {
  display: flex;
}

Vantagens do Flexbox

  • Layouts responsivos sem usar float ou positioning
  • Alinhamento e distribuição simples de espaço entre itens
  • Capacidade de redimensionar itens flexíveis para preencher o espaço disponível
  • Controle sobre a direção do layout (linha ou coluna)

Propriedades Principais

flex-direction: Define a direção dos itens (row, column, row-reverse, column-reverse)

justify-content: Alinha os itens ao longo do eixo principal (center, flex-start, flex-end, space-between, space-around)

align-items: Alinha os itens ao longo do eixo transversal (stretch, center, flex-start, flex-end, baseline)

flex-wrap: Controla se os itens devem quebrar para uma nova linha (nowrap, wrap, wrap-reverse)
