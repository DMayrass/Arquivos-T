Aprendendo Formulários HTML

Entenda as tags form, label, input, textarea e button

Explicação das Tags

A Tag <form>

O elemento <form> é um container para todos os elementos de um formulário. Ele define onde os dados serão enviados quando o usuário clicar em "Enviar".

<form action="/enviar.php" method="POST">
  <!-- elementos do formulário aqui -->
</form>

A Tag <label>

O elemento <label> define um rótulo para um elemento <input>. Melhora a acessibilidade e a usabilidade - ao clicar no rótulo, o campo de entrada é focado.

<label for="nome">Nome completo:</label>
<input type="text" id="nome" name="nome">

A Tag <input>

O elemento <input> é usado para criar campos de entrada de dados. O atributo "type" define que tipo de entrada será criada (text, email, password, etc.).

<input type="text"> <!-- Campo de texto -->
<input type="email"> <!-- Campo de email -->
<input type="password"> <!-- Campo de senha -->
<input type="checkbox"> <!-- Caixa de seleção -->

A Tag <textarea>

O elemento <textarea> define um campo para múltiplas linhas de texto, útil para comentários ou mensagens longas.

<textarea id="mensagem" name="mensagem" rows="4" cols="50">
Texto padrão pode ser colocado aqui.
</textarea>

A Tag <button>

O elemento <button> cria um botão clicável. Dentro de um formulário, o tipo padrão é "submit" (enviar).

<button type="submit">Enviar Formulário</button>
<button type="reset">Limpar Campos</button>
<button type="button">Botão Comum</button>

Pratique Aqui

Exercício Prático

Crie um formulário de cadastro com os seguintes campos:

  • Nome (texto)
  • E-mail (email)
  • Data de nascimento (data)
  • Gênero (seleção única)
  • Interesses (seleção múltipla)
  • Senha (password)
  • Botão de enviar

Dica: Use <input type="radio"> para seleção única e <input type="checkbox"> para seleção múltipla. Para agrupar opções, use <fieldset> e <legend>.

<fieldset>
  <legend>Gênero</legend>
  <input type="radio" id="masculino" name="genero" value="masculino">
  <label for="masculino">Masculino</label><br>
  <input type="radio" id="feminino" name="genero" value="feminino">
  <label for="feminino">Feminino</label>
</fieldset>

Desafio Avançado

Crie um formulário de pesquisa com:

  • Campo para nome
  • Seleção de faixa etária (dropdown)
  • Avaliação de 1 a 5 (radio buttons)
  • Sugestões (textarea)
  • Checkbox para receber novidades
  • Botões de enviar e limpar formulário

Dica: Use <select> com <option> para o dropdown de faixa etária. Para botão de limpar, use <button type="reset">.

<label for="idade">Faixa etária:</label>
<select id="idade" name="idade">
  <option value="18-25">18-25 anos</option>
  <option value="26-35">26-35 anos</option>
  <option value="36-45">36-45 anos</option>
  <option value="46+">46+ anos</option>
</select>
